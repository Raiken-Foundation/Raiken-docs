# Raiken Docs

Modern documentation site powered by Astro + Starlight, with a premium dark
theme, hero landing page, and feature cards.

## Quick start

```bash
npm install
npm run dev
```

Open `http://localhost:4321` to view the site.

## Customize

- Theme + landing page: `src/styles/theme.css`
- Homepage content: `src/content/docs/index.md`
- Sidebar: `astro.config.mjs`
